RAIL project requirements

System requirements
- Node.js 20.x
- npm 10.x
- PostgreSQL 14+ running locally or reachable over network
- Android Studio with Android emulator for mobile testing
- Expo CLI via npx (no global install required)

Backend runtime requirements
- @fastify/multipart ^10.0.0
- @prisma/adapter-pg ^7.7.0
- @prisma/client ^7.7.0
- bcrypt ^5.1.1
- dotenv ^17.4.2
- fastify ^5.6.1
- jsonwebtoken ^9.0.2
- pg ^8.20.0

- @types/node ^24.7.2
- @types/pg ^8.20.0

Environment variables required at repo root
- DATABASE_URL=postgresql://<db-user>@localhost:5432/<db-name>
- JWT_SECRET=<your-jwt-secret>

Project folders that need dependencies installed
- repo root: backend dependencies
- mobile/: Expo mobile dependencies

Files expected by the project
- .env at repo root
- prisma/schema.prisma
- mobile/app.json
- mobile/App.tsx

Optional but commonly needed
- Expo Go on device or emulator
- adb available through Android Studio SDK
